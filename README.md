# dotfiles

## Installation
  ```
  git clone https://github.com/msAlcantara/dotfiles $HOME/.dotfiles

  $HOME/.dotfiles/install <config> # $HOME/.dotfiles/install -h to list configs
  ```

## Usage
  ```
  dot <context> <cmd> <args>...  # dot tmux reload, for example
  ```

## Help
  ```
  dot -h
  ```
